# DevSecOps Secure Web Application

This project demonstrates the implementation of Secure SDLC and DevSecOps
principles using a simple Flask web application.

## Application Features
- User login
- Database storage (SQLite)
- Input handling

## Security Focus
- Protection against SQL Injection
- Input validation
- Secure coding practices

## DevSecOps Pipeline
- CI/CD using GitHub Actions
- SAST, SCA, and DAST security scans
- Shift Left security approach

## Tools Used
- Python (Flask)
- GitHub Actions
- OWASP security principles

## Shift Left Approach
Security is integrated early during requirements, design, and development phases
to detect vulnerabilities before deployment.
